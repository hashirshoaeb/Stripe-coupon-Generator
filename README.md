# Stripe Coupon Generator React App

## Getting Started

- make config.env file in [server/](./server)
- write stripe API as `STRIP_API`
- run server.js
    ```bash
     cd server
     npm install
     node server.js
    ```
- run react app
    ```bash
     cd ..
     npm install
     npm start
    ```
